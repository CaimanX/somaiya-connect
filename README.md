Somaiya Connect 
Live Demo:[https://somaiya-connect.web.app/](https://somaiya-connect.web.app/)

Overview
Somaiya Connect is a real-time, community-driven platform designed to enhance campus life. It centralizes lost & found, complaints, academic resources, and live campus "vibe" updates into one gamified ecosystem.

 Key Features
- Real-Time Feed: Lost & Found and Complaints with urgency levels.
- Gamification: Earn points by helping others and topping the leaderboard.
- Vibe Check: Crowdsourced live status of Canteens and Sports grounds.
- Security: Accountable user tracking via Google Auth (Future: @somaiya.edu restriction).
- Academic Hub:Peer-to-peer resource sharing with an upvote system.

 Tech Stack
- Frontend: React.js, CSS3 (with Dark Mode support)
- Backend: Firebase Authentication, Cloud Firestore
- Deployment: Firebase Hosting
